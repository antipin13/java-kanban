# Java Kanban Task Manager

Проект представляет собой доску для управления задачами с поддержкой различных типов хранения данных.

## О проекте

Kanban-доска - это приложение для эффективного управления задачами, позволяющее отслеживать их статус. Проект был разработан для повышения личной продуктивности и организации рабочих процессов.

## Функциональность

- Создание, редактирование и удаление задач
- Изменение статуса задач (NEW, IN_PROGRESS, DONE)
- Поддержка различных типов задач (стандартные, эпики, подзадачи)
- Два типа хранения данных: in-memory и CSV-файлы

## Технологии и инструменты

- **Java Core** - основной язык разработки
- **CSV-файлы** - для хранения данных
- **JUnit** - для модульного тестирования
- **Collections Framework** - для работы с данными в памяти
- **File I/O** - для операций с файловой системой

## Архитектура

Проект реализован с использованием объектно-ориентированного подхода:

- `TaskManager` - основной интерфейс менеджера задач
- `InMemoryTaskManager` - реализация с хранением в оперативной памяти
- `FileBackedTaskManager` - реализация с сохранением в CSV-файл
- `Task`, `Epic`, `Subtask` - модели данных задач
- `TaskStatus` - перечисление статусов задач

## Результаты

Разработано гибкое и наглядное приложение для управления задачами, которое:
- Повышает личную эффективность в организации рабочих процессов
- Поддерживает различные сценарии использования благодаря гибкой системе хранения данных
